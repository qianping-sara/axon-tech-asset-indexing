# Getting Started with Axon

Welcome to Axon, your comprehensive automation asset management platform.

## What is Axon?

Axon is a powerful platform designed to help you discover, organize, and manage automation assets across your organization. Whether you're looking for code components, APIs, workflows, or best practices, Axon makes it easy to find exactly what you need.

## Key Features

### Browse by Category
Explore automation assets organized by type:
- **Code & Components** - Scripts, libraries, frameworks, and reusable code modules
- **Services & APIs** - REST APIs, GraphQL endpoints, microservices, and web services
- **AI/ML Services** - Machine learning models, LLM services, AI agents, and ML pipelines
- **Automation & Workflows** - RPA bots, workflows, processes, and automation scripts
- **Data & Analytics** - Data products, schemas, pipelines, and analytics solutions
- **Architecture & Governance** - Architectures, patterns, standards, policies, and governance frameworks
- **Knowledge & Practices** - SOPs, playbooks, tutorials, guides, and best practices

### Search & Discover
- Use the search functionality to find assets by name or keyword
- Discover recently updated assets on the Discover page
- Filter assets by category and tags

### Asset Details
Each asset includes:
- Detailed description
- Associated tags for easy filtering
- Asset count and metadata
- Direct links to resources

## Getting Started

1. **Browse Categories** - Start by exploring assets in the "Browse by Category" section on the home page
2. **Search** - Use the search bar to find specific assets
3. **Discover** - Check out recently updated assets on the Discover page
4. **View Details** - Click on any asset to see more information

## Tips & Tricks

- Use tags to filter assets by specific technologies or use cases
- Check the Discover page regularly for new and updated assets
- Bookmark frequently used assets for quick access
